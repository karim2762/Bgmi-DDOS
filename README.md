## ✨ Features

- 🔐 Admin authentication system
- 🤖 Telegram Bot integration
- 👥 Support for multiple administrators
- 📅 User access with configurable expiry periods
- 🆔 Access management based on Telegram User ID
- ⚙️ Easy configuration and deployment
- 🛡️ Permission-based command handling
- 📊 Organized project structure for future expansion

## ⚙️ Configuration

Before running the project, configure:

- Telegram Bot Token
- Administrator IDs
- Authorized user records
- Access duration settings
- Other runtime options in the configuration files

---

## 🔑 Access Management

The project supports:

- ✅ Multiple Admin Accounts
- ✅ Telegram ID based authorization
- ✅ Custom validity periods for users
- ✅ Controlled access through the bot interface
- ✅ Expandable permission management system
